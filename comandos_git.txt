git init .    -> Cria repo git na pasta atual
git status   -> Mostra o status do trabalho
git add -> Adiciona arquivo para ser commitado
git commit -m "msg" -> Faz commit do que estava pra ser commitado
git log -> Mostra todo histórico de commit

FLUXO DE TRABALHO
1. criar diretório
2. iniciar o git dentro do diretório
3. edita ou cria arquivos
4. git status
5. git add nome_do_arquivo
6. git commit --m "mensagem"
7. git status
8. volta para o passo 3
